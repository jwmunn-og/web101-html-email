Email Template
Getting Started

Customize this template and use it as a base to build your own responsive emails. It’s best to rely on web safe fonts for your text as @font-face and Google font links aren’t well supported.

Style your text so that it is easy to read. Using <div> <span> & <p> tags allow you to call out text with special styles. You can even style your Call to Action buttons with CSS so they still show up even when images are turned off.
